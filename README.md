# Dashboard de Controle Financeiro

## 📌 Descrição do Projeto
Desenvolvimento de um dashboard interativo para análise de dados financeiros, com foco no acompanhamento de receitas, despesas e saldo ao longo do tempo.
O projeto tem como objetivo transformar dados brutos em informações visuais claras, auxiliando na tomada de decisão.

---

## Principais Funcionalidades

- Indicadores de desempenho (KPIs):
  - Total de Receitas
  - Total de Despesas
  - Saldo Atual

- Análise de despesas por período (mensal)
- Distribuição de gastos por categoria
- Filtros interativos por mês e categoria

## Tecnologias Utilizadas

- Power BI (visualização de dados)
- MySQL (armazenamento e consulta de dados)
- DAX (criação de métricas)
- Power Query (tratamento de dados)

## Principais Análises

- Identificação dos períodos com maior volume de gastos  
- Análise da distribuição de despesas por categoria  
- Comparação entre receitas e despesas ao longo do tempo  

## 📂 Estrutura do Projeto

- `dashboard-controle-gastos.pbix` → Dashboard desenvolvido no Power BI  
- `dados/` → Base de dados utilizada  
- `imagens/` → Visual do dashboard  
- `fundo/` → Recursos visuais
